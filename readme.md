# 制作アプリのタイトル
- ドットアンドボックス

## 制作アプリの説明
- 一般的なドットアンドボックスのゲームです
- ルールも昔からあるものに準拠してます
- 交互に線を引いて、正方形を完成させたときのプレイヤーに１点入ります
- 線を引いて完成させると続けて線を引けます
- すべての線を引き終わったら獲得した点数で勝敗を決めます

## 工夫した点・こだわった点
- オンライン対戦が実現するようにサーバーを通してやり取りするデータに何が必要かは自力で考えました
- AIの使い方も考えて、自分でこういった処理を行うと考えて、やってみてわからないときだけ質問するようにしました
- 一つ一つの必要な処理を組み合わせて、一通り実現したい処理が完成するまではほとんど自分の力で完走できるように、というところは気を付けました（おんぶにだっこにならないように、何の処理が必要なのかを自分で考えられるように）

## 次回トライしたこと（または機能）
- 大量のデータを扱って、集計したり統計データを出したり、一覧にしたり、そういうこともやってみたいです
